# Security beleid
Wanneer je een bug of kwetsbaarheid denkt te hebben gevonden in de code, deze graag melden door te mailen naar [dcc@de-alliantie.nl](mailto:dcc@de-alliantie.nl).

**Rapporteer bugs dus *niet* met behulp van publieke communicatie zoals issues, pull requests, forums etc. in verband met security.**

Voeg aan je melding zoveel informatie toe als mogelijk, zoals:
- Type issue
- Impact van de issue: hoe kan de issue gebruikt worden?
- Stappenplan voor het reproduceren van de issue
- Locatie van de issue in de broncode
- Code om te issue te exploiten

Deze informatie helpt ons om het probleem sneller op te lossen.
